For-Sale Extractor — CCARCS Dashboard

Run these bookmarklets on any aviation listing site in your own browser. Your browser session bypasses bot-detection and CAPTCHAs automatically.

Step 1 — Diagnose (run this first if the extractor fails)

Run this on the listing page to see what link patterns the site uses:

🔍 Diagnose Links Opens a box with sample links — copy and share so the extractor can be fixed for this site.

Step 2 — Extract all listings

📋 Extract All Listings Visit a search results page, click this. Crawls every detail page automatically and downloads forsale_manual.json.

Supported sites — Canada search URLs

Usage

1

Open one of the Canada search URLs above in your browser.

2

Click Extract All Listings. A progress counter appears top-right. Takes ~1 min for 100 listings.

3

If it says "No listing links found", click Diagnose Links and share the output so the URL filter can be fixed.

4

Merge results: run the extractor on each site, then combine the downloaded JSON files before dropping into d:\gilead\ClaudeProjects\. Don't overwrite — merge, so listings from all sites are preserved.

5

Run the update script (or let the daily GitHub Action do it).
