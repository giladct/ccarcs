Option A — Local (Windows)

  1. Double-click update_data.bat in the project folder

Option B — Python (any OS)

  1. Install once: pip install requests
  2. Run: python update_data.py
  3. Reload this page (F5)

Option C — GitHub Actions (no local files needed)

  1. Click the button below to open GitHub Actions
  2. Click Run workflow → Run workflow
  3. Wait ~2 minutes, then reload this page (F5)
